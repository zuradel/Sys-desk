# 🖥️ SysDesk — Server Monitor with Live2D Girls' Frontline

An anime Live2D character lives inside your Home Assistant card, watches your CPU, RAM, disk, and temperatures in real time, and **speaks every alert out loud** — on a loop — until the issue is resolved.

## ✨ Highlights

- 🔫 **9 Girls' Frontline characters** — switchable with ◀ ▶ buttons
- 🚨 **Threshold alerts** — CPU, RAM, Disk, CPU Temp, Drive Temp (warn + critical)
- 🔁 **Auto-TTS loop** — spoken warning repeats until metric returns to safe range
- 🖥️ **Multi-server** — Home Assistant · pfSense · Frigate · NAS
- 📡 **Service & AP monitoring** — AdGuard, WiFi APs, any switch or sensor
- 📌 **Mini + Pin Mode** — floats above your entire dashboard across all views
- 🎛️ **Visual editor** — no YAML required
- 🌐 **i18n** — Vietnamese 🇻🇳 & English 🇬🇧

## 📦 Installation

1. Add this repository to HACS as a **Dashboard** plugin
2. Search **SysDesk** → Install
3. Hard-reload (`Ctrl+Shift+R`)
4. Add to your dashboard: `type: custom:sys-desk`
